# Social Media Platform

## Description
Java project that simulates a simple social media platform.

## Classes
- User – represents a platform user
- Post – represents a post created by a user
- SocialMedia – publishes posts

## Features
- Object-oriented programming (OOP)
- Constructors
- Getters and setters
- Object comparison using equals()

## Technologies
- Java
- IntelliJ IDEA
